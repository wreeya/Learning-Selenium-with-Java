# Learning-Selenium-with-Java
This repository contains my practice code while learning Selenium WebDriver with Java.
Currently, it contains a simple automation script that:
- Opens Chrome browser
- Navigates to SauceDemo website
- Enters username and password
- Clicks the locatorsexample button

Tools used:
- Java
- Selenium WebDriver
- ChromeDriver
- Maven

This project is for learning and practice purposes.
